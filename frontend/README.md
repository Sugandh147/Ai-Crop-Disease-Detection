# 🎨 Kisan AI – Next.js 16 / React 19 Frontend

This repository contains the Next.js frontend web application for **Kisan AI – AI Crop Disease Detection & Agricultural Advisory System**.

---

## 🚀 Getting Started

First, ensure the FastAPI backend is running on `http://localhost:8000`.

Then start the frontend development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to launch the web application.

---

## 🌟 Key Capabilities

- **Drag-and-Drop Dropzone:** Simple upload interface with instant leaf previews and upload progress indicator.
- **4-Tab Diagnostic Card:** Renders **Overview**, **Symptoms & Causes**, **Treatment Plan**, and **Action Checklist**.
- **23-Language Selector:** Instant dynamic translation across 23 regional Indian languages.
- **Hydration-Safe Persistence:** Safely persists language choice in `localStorage` without React SSR hydration errors.
